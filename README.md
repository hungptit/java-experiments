# java-experiments
This repository contains my random experiments with Java.

# How to 

## Execute benchmarks ##

**Run all benchmarks**

``` shell
java -jar target/experiments-0.0.1-SNAPSHOT.jar
```

**Run benchmarks that match a given pattern**

``` shell
java -jar target/experiments-0.0.1-SNAPSHOT.jar MapDiff
```
